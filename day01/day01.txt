一.Linux常用命令行
Shell命令(linux的操作方式)
    1.pwd,cd,ls
        - pwd:print working directory
          显示当前工作目录/路径
        - cd:change directory
          改变工作路径
          使用:cd 目标文件夹/目录
               cd .         目标文件夹/目录
               cd ..        回到上一级
               cd ./a       进入当前文件下的a文件夹中
               cd ../a      进入上一级文件夹下的a文件夹中
               绝对路径
               cd /d/app    进入到d盘下的app文件夹中
               cd ~         进入到当前登录用户的主目录
        - ls:list
          ls 显示当前文件夹中所有非隐藏文件和目录
          linux下隐藏文件或目录的方式:以.开头文件或目录为隐藏

          ls -a     显示所有包括隐藏的
          ls -lt    以列表形式展示详细信息并按照时间顺序排列(升序)
          ls 路径    显示指定目录中的内容
    2.新建文件夹
        - mkdir (make directory)
          - mkdir 目录路径
            mkdir test 新建一个test文件夹
            mkdir ./test/a 在test文件夹新建一个a文件夹(没有文件就不会创建成功)
          - mkdir -p 目录路径 如果父级目录不存在,则先创建父级目录
            mkdir -p /a/b/c 没有a创建a,创建b,创建c(先创建父级目录,再创建自己目录)
          - mkdir a b 同时创建多个目录
            mkdir app/a app/b app/c

            如果文件夹已经存在,则报错
    3.创建文件
        - touch 
          - touch 1.txt 创建1.txt文件
            touch app/1.txt
            touch app/.txt(可以创建.开头的文件)

            如果touch文件已经存在,不会覆盖原来的文件,而是修改原来文件的最后访问时间
            touch还可以修改目录的最后访问时间
            touch目录名

    4.echo 
        - 将数据写入到指定文件,如果文件不存在,就创建
          echo 数据 > 文件名 (重新写会覆盖)
          echo 数据 >> 文件名 (追加模式->两行)
    5.vi
        vi 文件名
        :wq
        使用命令后,命令行会进入文本编辑界面
        此界面分为三种模式:
        命令模式:
          vi命令使用后进入的模式.可以输入a/i进入输入模式,也可以输入:进入低行模式
        输入模式:
          此模式下可以向文件中输入数据,输入完成后按ESC退出输入模式,进入命令模式
        低行模式:
          :wq 保存并退出
          :q 不保存,直接退出
          :w 保存,单不退出
          :q! 不保存,强制退出
    6.cat
        - catch 获取某个文件的内容
          cat 文件名
          cat 1.txt
    7.删除文件夹
        - rmdir (remove directory)
          redir 目录名 删除目前(永久删除)
          只能删除空目录,如果目录中有其他内容,则删除失败
          mkdir a 
          rmdir a
          
          mkdir -p a/b
          rmdir a(报错:command not found)

    8.删除文件
        - rm
          rm 文件名

          如果要删除非空目录
          rm -rf 目录名
          touch lyf.txt
          rm -rf lyf.txt

二.git命令
  注:所有的git命令都是以git开头的

  1.配置用户信息(只需要配置一次)
    git config --global user.name   "用户名"
    git config --global user.email  "邮箱"
               --system
  2.初始化本地仓库
    git init
    进入到目标文件夹中使用命令git init
    初始化成功后,目标文件夹中会自动生成一个隐藏的.git文件夹,同时git bash命令行路径后面会多一个(master)

    .git文件夹就是本地仓库的版本库
  3.查看本地仓库的状态
    git status
    三大区域:
      工作区:     仓库所在文件夹,保存文件的地方
      暂存区:     版本库中暂存
      分支所在区: HEAD指向分支
    四种状态:
      modified:   文件有修改,但是还没有暂存
      untracked:  新建的文件,还没有被git管理控制住
      staged:     文件被暂存,还没有被提交
      unmodified: 文件被提交
  4.提交
    提交分两个步骤:
    1) 从工作区提交道暂存区
        git add 文件名
    2) 从暂存区提交到master分支
        git commit -m "提交的信息"

        git checkout -- a:撤销工作区中所有的修改

  步骤:
  1.新建一个文件
  2.使用git add命令提交到暂存区
  3.使用git commit -m命令提交到master分支
  *每使用一个命令后都要用一下git status查看状态
